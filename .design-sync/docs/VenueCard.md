category: Venues

VenueCard is a clickable summary card for a venue: name, a color-coded surface-type chip (Ice / Turf / Court / Field), address and location, and a row of outlined chips for capacity, visibility (Public / League / Team), owning team or league, and an Inactive flag. Inactive venues render dimmed. Clicking navigates to the venue detail page.

```tsx
<VenueCard venue={venue} />
```
